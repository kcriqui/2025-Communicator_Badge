# Fish

A simple aquarium simulation app for the badge.

This app displays a number of colorful fish swimming around in a virtual tank with a sandy bottom. Bubbles periodically rise from the bottom to the top of the screen. This app is purely for visual enjoyment and can act as a screensaver.

## Controls

- Press any key to exit the app and return to the menu.

## Installation Instructions

Copy `fish.py` to your `firmware/badge/apps` folder and modify `firmware/badge/main.py` to add "Fish" to your User Apps menu.
